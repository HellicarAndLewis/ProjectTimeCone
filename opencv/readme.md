ProjectTimeCone/opencv/Debug
ProjectTimeCone/opencv/Debug/bin
ProjectTimeCone/opencv/Debug/lib
ProjectTimeCone/opencv/Release
ProjectTimeCone/opencv/Release/bin
ProjectTimeCone/opencv/Release/lib
